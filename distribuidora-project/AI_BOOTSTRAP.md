# 🚀 Prompt para Iniciar un Nuevo Proyecto (AI Bootstrap)

Copia y pega este prompt cuando inicies un proyecto nuevo desde cero con tu agente:

---

**PROMPT:**

"Hola, voy a iniciar un nuevo proyecto llamado **[NOMBRE_DEL_PROYECTO]**.
Quiero establecer un sistema de memoria y contexto persistente desde el principio.

Por favor, realiza lo siguiente:

1.  **Entrevista Inicial**: Hazme 3-5 preguntas clave para entender el objetivo, el stack tecnológico preferido y las reglas de diseño.
2.  **Inicialización de Archivos**: Una vez tengas mis respuestas, crea inmediatamente dos archivos en la raíz:
    *   `AI_CONTEXT.md`: Con la identidad del proyecto, stack, reglas y arquitectura (usa el formato estándar de "Pasaporte de Proyecto").
    *   `TASKS.md`: Con la lista de tareas iniciales (Planificación, Configuración Base, etc.).
3.  **Regla de Mantenimiento**: A partir de ese momento, tu regla principal será actualizar estos archivos antes de finalizar cada sesión o tarea importante."

---

### ¿Por qué funciona este prompt?
Este prompt convierte al agente en un "Project Manager" desde el segundo cero. En lugar de empezar a tirar código al azar, le obligas a:
1.  Entender el problema (Entrevista).
2.  Crear su propia "memoria" (`AI_CONTEXT.md`).
3.  Planificar (`TASKS.md`).
